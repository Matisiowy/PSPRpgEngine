# Roadmap

## M0 - Foundation

- PC editor shell;
- versioned JSON project;
- scene preview;
- PSP runtime boot and GU test scene.

## M1 - First end-to-end build

- asset compiler CLI;
- binary scene format;
- editor Build for PSP command;
- the same entity visible in editor and PSP runtime.

## M2 - Map authoring

- tileset import;
- ground, objects and collision layers;
- tile painting tools;
- camera and map renderer on PSP.

## M3 - Playable prototype

- PSP input;
- player entity and animation;
- tile collision;
- map transitions and triggers.

## M4 - RPG toolset

- NPC and dialogue editors;
- event graph;
- items and inventory;
- quests and conditions;
- save-game system.

## M5 - Production pipeline

- texture/audio conversion;
- package compression;
- memory-budget diagnostics;
- project templates and distributable PSP builds.
